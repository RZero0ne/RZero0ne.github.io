# BUUCTF_MISC(21-35)

# 21. 隐藏的钥匙

图片拖进 010，搜 flag，发现了一段 base64 密文

得到 flag

# 22. 另外一个世界

拖到 010 最后发现一段 ascii 码

对照码表转换成字符

flag

# 23. 神秘龙卷风

发现加密 zip，根据题目提示 4 位数字密码，

解压得到 txt

brainfuck 加密：https://www.splitbrain.org/services/ook

brainfuck to text 出 flag

# 24.FLAG

打开得到 png 文件

用 zsteg 分析发现有个 zip 隐藏其中，用命令把 zip 提取出来

zip 拖到 windows 里面解压出来了一个 1 文件

用 PE 查到文件是 64 位，拖进 ida 里面看到 flag

hctf

# 25. 数据包中的线索

一开始分析 tcp 流，发现内容显示不完全，分析 http 流

在第 7 流中发现一段 base64 密文，复制下来解密，被识别出 jfif 文件自动转换成 jpg 文件

打开发现 flag

# 26. 假如给我三天光明

打开图片发现盲文

对照转换表

解密 kmdonowg

拿到 zip 密码，得到一个 music.wav 文件

摩斯电码解密出 flag

# 27. 后门查杀

用 d 盾扫描出来

打开 include.php 找到 md5

6ac45fb83b3bc355c024f5034b947dd3

# 28.webshell 后门

进入 zp.php 找到 flag

# 29. 来首歌吧

#

摩斯电码解密

5BC925649CB0188F52E617D70929191C

# 30. 面具下的 flag

先用 binwalk 分离出 zip 文件

发现需要密码，把 zip 拖进 0101 里发现是伪加密，把 09 改成 00，得到 flag.vmdk 文件，

用 linux 里自带的 7z 解压，得到 2 个文件，

第一个文件里有个 NUL，cat 查看，brainfuck 解密得到第一段 flag，

第二个文件里 cat 出 Ook！密文，解密得到第二段 flag

flag

# 31. 荷兰宽带数据泄露

用 RouterPassView 查找 username 和 password，flag 是 <Username val=053700357621 />

flag

# 32. 九连环

binwalk 分离出来了一个 zip，发现是伪加密，改成 14 00 00 08

解压得到一个 jpg 和一个加密 zip，空密码

txt 里面就是 zip 的密码：flag

# 32. 被劫持的神秘礼物

导出 http，有三个文件，login 里面有账号密码，合并起来：adminaadminb

# 33.[BJDCTF2020] 认真你就输了

binwalk，foremost 出来了 zip，找到文件里的 flag.txt

# 34. 被偷走的文件

在 tcp 追踪流中，在第 3 流中发现 rar 文件

binwalk 出来一个 rar 文件

爆破出来 4 位密码

flag

# 35.[BJDCTF2020] 藏藏藏

foremost 出来了一个 zip 文件，解压出来一个 jpg

扫描出来

flag
