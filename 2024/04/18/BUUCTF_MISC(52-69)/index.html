# 54.[SWPU2019] 我有一只马里奥

运行 exe 发现生成了一个 1.txt

ntfs 交换数据流

NTFS 交换数据流（Alternate
DataStreams，简称 ADS）是 NTFS 磁盘格式的一个特性，在 NTFS 文件系统下，每个文件都可以存在多个数据流。通俗的理解，就是其它文件可以 "寄宿" 在某个文件身上。利用 ADS 数据流，我们可以做很多有趣的事情，同时对于企业安全来说这类问题也隐藏着巨大的风险。

打开 cmd，输入 notepad 1.txt:flag.txt

得到 flag
