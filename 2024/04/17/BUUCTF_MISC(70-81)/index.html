这段代码是一个简单的加密脚本，主要功能是对一个字符串（标记为 flag）进行加密，然后将加密后的结果以列表形式返回。加密过程如下：

  1. 定义一个名为 encode 的加密函数，该函数不接受任何参数。
  2. 在函数内部，有一个名为 flag 的字符串，这里用’*' 代替，实际使用时应替换为具体的字符串。
  3. 初始化一个空列表 ciphertext，用于存储加密后的字符。
  4. 使用 for 循环遍历 flag 字符串中的每个字符。
    对于每个字符，首先计算它的 ASCII 值与其索引（i）的异或（XOR）结果，然后将该结果转换回字符（s）。
    接下来，根据字符的索引是奇数还是偶数，将字符 s 的 ASCII 值分别加上 10 或减去 10。
    将处理后的 ASCII 值转换为字符串，并添加到 ciphertext 列表中。
  5. 最后，将 ciphertext 列表反转，并返回。
    脚本的最后部分给出了一个名为 ciphertext 的列表，这是使用上述加密函数处理某个特定 flag 字符串后的输出结果。
    要理解或逆向这个加密过程，需要关注几个关键点：
    异或操作是可逆的，即如果你知道原始字符和索引，你可以通过再次执行异或操作来恢复原始字符。
    加 10 或减 10 的操作也是可逆的，只需相反操作即可恢复原始值。

if ( v4 == 122 && String[3] == 120 && String[5] == 122 && String[4] == 121 )

# 80.[安洵杯 2019] 吹着贝斯扫二维码

下载压缩包解压得到这么一些文件，可以看到一堆未知类型文件和一个 flag.zip

把这些没有拓展名的文件用 010Editor 打开，发现都是 jpg 类型文件

需要给这些文件添加拓展名，直接使用 cmd 下的 ren 命令批量重命名，添加.jpg 拓展名。

命令：ren * *.jpg
// 这条命令会把此目录下所有文件重命名，所以使用的时候注意目录位置
// 还有注意这里的 flag.zip 文件也被重命名，改回.zip 后缀就行
